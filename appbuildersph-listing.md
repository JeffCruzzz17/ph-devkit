## Full description

PH DevKit is a free toolkit for Filipino developers building Philippine-ready web applications. It includes reusable React components, TypeScript utilities, backend validation examples, SQL schema patterns, and practical docs for common local app requirements.

The MVP includes a Philippine address selector, peso formatter, Philippine mobile number validator, Laravel validation examples, .NET Web API examples, SQL schema examples, and a live component playground.

Built for makers creating SaaS apps, booking systems, dashboards, directories, business tools, school systems, finance tools, and internal admin apps for the Philippine market.

## Founder comment

Hey builders, I am Jeff. I built PH DevKit because many Philippine apps need the same basics: address forms, peso formatting, mobile validation, auth/RBAC patterns, audit logs, and clean API examples.
