# Security Policy

## Supported versions

PH DevKit is currently in MVP development.

| Version | Supported |
|---|---|
| 0.1.x | Yes |

## Reporting a vulnerability

Please do not open a public issue for security vulnerabilities.

Report suspected vulnerabilities by contacting the maintainer privately.

Include:

- Affected package, component, or example
- Steps to reproduce
- Potential impact
- Suggested fix, if available

## Scope

Security reports are welcome for:

- `@ph-devkit/core`
- `@ph-devkit/react`
- Website playground
- Laravel examples
- .NET examples
- SQL examples
- GitHub Actions / CI configuration

## Out of scope

The demo address dataset is intentionally small and is not production PSGC data.

## Maintainer response target

For MVP, the maintainer will try to acknowledge valid vulnerability reports within 7 days.
